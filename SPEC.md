# 김간호 하늘색 바지

## 개요
김간호(플레이어) 스프라이트의 하의 색을 기존 살구빛(`#e8bcb4`)에서 하늘색 스크럽 팬츠 톤으로 변경한다. 수간호사(올-화이트 유니폼)와의 색상 대비를 강화하고, 흰 상의 + 코럴 십자 + 번 머리로 구성된 플레이어 실루엣에 세 번째 대비 색을 더해 캐릭터 정체성을 명확히 한다.

## 변경 유형
**디자인** — 단일 팔레트 값(`P` 키) 교체. 로직·레이아웃·픽셀 배치 변경 없음.

## 디자인 언어 & 의도
하늘색은 스크럽 팬츠를 연상시키는 의료 복장 언어로, 흰 상의 위에서 수간호사의 올-화이트 유니폼과 즉각 구분된다. 흑갈 번 + 흰 상의 + 코럴 십자의 "따뜻한 축"에 차가운 톤 하의가 들어가면서 치비 비율 캐릭터의 상·하체 경계가 단일 픽셀 두께에서도 분명히 읽힌다.

## 변경 범위

### assets/css/game.css 변경사항

1. `:root` 블록에 추가:
   ```
   --nurse-pants: #9ec9e8;
   ```
2. `html.light` 블록에 추가:
   ```
   --nurse-pants: #7fb5d8;
   ```
라이트 테마 값은 밝은 바닥 타일 위에서 콘트라스트 유지를 위해 한 톤 어둡게.

### assets/js/game.js 변경사항

`getNursePalette()`의 `P` 키 한 줄 교체:
- **변경 전**: `'P': '#e8bcb4',`
- **변경 후**: `'P': readVar('--nurse-pants', '#9ec9e8'),`

`readVar`는 같은 함수 내에 이미 선언되어 있으므로 재정의 불필요. `nursePaletteCache` 무효화는 테마 토글 핸들러에 이미 포함되어 있어 추가 작업 없음.

그 외 `getNursePalette`의 9개 키(`S/W/C/B/E/L/R/M/H/b`) 한 줄도 수정하지 않는다.

## 기능 상세

### 기능 1: 김간호 바지 색상 하늘색화 (테마 반응형)
- **설명**: 김간호 스프라이트 하의 영역(`P`, 행 15/16/17)이 다크에서 `#9ec9e8`, 라이트에서 `#7fb5d8`로 렌더링.
- **구현 위치**:
  - `assets/css/game.css` `:root`, `html.light` 두 블록에 `--nurse-pants` 정의
  - `assets/js/game.js` `getNursePalette()` `P` 키를 `readVar('--nurse-pants', ...)` 로
- **세부 영향 범위**:
  - 행 15 `....PPPPPPPP....` (8px)
  - 행 16/17 `....PPP..PPP....` (각 6px)
  - 방향 무관(`down/up/left/right` 모두 동일)
  - 걷기 프레임은 행 18/19만 변경 → 바지 영역 영향 없음

## 주의사항

- **캐시 무효화 이미 구현**: `nursePaletteCache = null`이 테마 토글 핸들러에 존재. 추가 코드 불필요.
- **수간호사 영향 없음**: `drawNurseChief`는 `getChiefPalette`를 사용하고 `P` 키를 참조하지 않음.
- **프리뷰 캔버스 포함**: `drawNurse(...)` 시연 호출도 동일 팔레트 공유 → 자동 반영.
- **하드코딩 정책 일관성**: 기존 "테마에 따라 명도가 바뀌는 색(H/b)만 CSS 변수" 기준과 부합. `P`도 테마 반응형이므로 변수 측으로 이동이 자연스러움. 나머지 키는 기존대로 하드코딩 유지.
- **접근성/보안**: 색상만 바뀜. `prefers-reduced-motion`/XSS 관련 추가 작업 없음.

## 검증 기준
1. 다크 테마에서 김간호 바지 = 연한 하늘색(`#9ec9e8`)
2. 라이트 테마 전환 시 = 살짝 진한 하늘색(`#7fb5d8`)
3. 김간호의 번/얼굴/상의/코럴십자/신발 색 **변화 없음**
4. 수간호사 스프라이트 픽셀 한 개도 색 변화 없음
5. 걷기 애니메이션 및 방향 전환 정상 동작
6. 게임 로직(충돌/점수) 영향 없음
