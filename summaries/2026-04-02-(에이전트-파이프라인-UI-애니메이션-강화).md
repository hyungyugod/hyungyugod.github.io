# 📌 2. UI 인터랙션·애니메이션 대폭 강화

## 2-1. 카드 호버 효과 고도화
> 💡 conic-gradient 회전 보더와 플랫폼별 글로우 색상으로 카드에 시각적 개성 부여

### 1) 회전 보더 애니메이션
- `@property --angle` CSS Houdini로 각도 커스텀 프로퍼티 등록
- `conic-gradient(from var(--angle))` 기반 보더가 4초 주기로 회전
- 호버 시에만 opacity 1로 전환되어 자연스럽게 나타남

### 2) 플랫폼별 글로우 색상
- Velog(초록), Brunch(회색), GitHub(흰색), Melon(녹색), SoundCloud(주황) 5종 글로우 변수 추가
- `:has()` 셀렉터로 카드 내 아이콘 클래스를 감지하여 플랫폼에 맞는 `box-shadow` 적용
- 다크/라이트 테마별 글로우 투명도를 각각 설정

### 3) 카드 미세 기울기·지연 시간 조정
- 홀짝 카드에 각각 `rotate(0.5deg)` / `rotate(-0.5deg)` 미세 기울기 부여
- 카드 등장 지연 시간을 0.08s → 0.12s 간격으로 넓혀 시차 효과 강조

## 2-2. 모달 시차 진입·닫기 애니메이션
> 💡 모달 내부 요소들이 순차적으로 나타나고, 닫힐 때는 축소되며 사라지는 연출 추가

### 1) 시차 진입 애니메이션
- 모달 내부 요소(사진, 이름, 영문명, 구분선, 정보 항목)를 기본 `opacity: 0`으로 숨김
- `is-open` 클래스 추가 시 각 요소에 0.08s 간격의 `transition-delay`로 순차 등장
- 구분선은 `scaleX(0) → scaleX(1)`로 가로 펼침 효과 적용

### 2) 닫기 애니메이션 구현
- `is-closing` 클래스 추가 → 모달 박스가 `scale(0.95)`로 축소되며 투명해짐
- 오버레이도 `opacity: 0`으로 동시 페이드아웃
- JS에서 `transitionend` 이벤트 감지 후 클래스 제거, 500ms 폴백 타임아웃 안전장치 포함

### 3) 모달 배경 강화
- 오버레이 블러를 6px → 12px로 증가시켜 배경 분리감 향상
- `radial-gradient` 후광 효과 추가하여 모달 중심부에 브랜드 색상 은은하게 발산

## 2-3. 히어로·프로필 영역 애니메이션
> 💡 배경 그라디언트 이동과 아바타 부유 효과로 정적이던 히어로에 생동감 추가

### 1) 배경 그라디언트 시프트
- `background-size: 400% 400%`로 확대 후 `gradient-shift` 키프레임으로 15초 주기 이동
- 다크/라이트 테마 히어로 배경 모두 적용

### 2) 프로필 아바타 플로트
- `float-gentle` 키프레임으로 6초 주기 상하 6px 부유 애니메이션
- 타이핑 텍스트에 `display: inline-block`, `min-width: 1ch` 추가하여 깜빡임 안정화

## 2-4. 마그네틱 버튼·스프링 전환 곡선
> 💡 마우스 위치를 추적하는 마그네틱 효과와 탄성 바운스 곡선으로 인터랙션 질감 향상

### 1) 마그네틱 버튼 효과
- `initMagneticButtons` 함수 신규 추가 — 프로필 버튼과 카테고리 버튼에 적용
- 마우스 좌표 기반으로 버튼이 0.3배 비율로 미세 이동, 마우스 떠나면 원위치
- `hover: none`(터치 기기) 및 `prefers-reduced-motion` 환경에서는 자동 비활성화

### 2) 스프링 바운스 곡선 적용
- `--spring-bounce: cubic-bezier(0.34, 1.56, 0.64, 1)` 커스텀 프로퍼티 추가
- 프로필 버튼, 카테고리 탭, 모달 박스의 `transform` 트랜지션에 적용하여 탄성감 부여

## 2-5. 안정성 및 접근성 개선
> 💡 초기화 에러 격리와 모션 감소 대응을 확장하여 안정적인 사용자 경험 보장

### 1) safeInit 래퍼 도입
- 모든 초기화 함수를 개별 try-catch로 감싸는 `safeInit` 유틸리티 함수 추가
- 하나의 초기화가 실패해도 나머지 기능은 정상 동작하도록 에러 격리
- 실패 시 `console.warn`으로 어떤 함수가 실패했는지 디버그 로그 출력

### 2) prefers-reduced-motion 대응 확장
- 히어로 배경 `gradient-shift`, 아바타 `float` 애니메이션을 모션 감소 시 비활성화
- 카드 회전 보더(`::before`) 애니메이션도 `opacity: 0`으로 숨김
- 섹션 라벨의 `clip-path` 효과도 모션 감소 환경에서 제거

### 3) category-label 마진 미세 조정
- `margin-bottom: 16px`을 `margin-top: 8px` + `margin-bottom: 8px`으로 분리하여 상하 균형 조정
